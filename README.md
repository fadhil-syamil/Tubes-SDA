# Aplikasi Rekomendasi Wisata Bandung

Aplikasi ini menggunakan algoritma **Dijkstra** untuk menghitung rute terpendek antar tempat wisata di Bandung. Proyek ini menggabungkan performa **C++** untuk logika perhitungan dan fleksibilitas **Python** untuk antarmuka pengguna.

## Prasyarat
Sebelum menjalankan aplikasi, pastikan Anda telah menginstal:
1. **Python 3.x**
2. **C++ Compiler** (G++ disarankan)
